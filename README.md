# Greeting App

## Description
A simple modern greeting web application built using HTML, CSS, and JavaScript with the help of AI.

## Features
- Display `Enter Your Name`
- Input field with placeholder `Type your name here`
- `Greet` button
- Show `Hello` when button is clicked
- Random background animation on each click (`confetti`, `glowing effect`, `color burst`)
- Ensure only one animation runs at a time
- Centered and clean UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
Option 1: Open `index.html` directly in your browser.

Option 2: Run a local server using:

```bash
python -m http.server 5500
```

Then open `http://localhost:5500`.
