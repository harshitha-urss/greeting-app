# Greeting App

## What it does

When you load the page, you’ll see:
- A label: `Enter Your Name`
- A text input: placeholder `Type your name here`
- A button: `Greet`

Each time you click `Greet`, the app shows `Hello` and randomly triggers one of three background animations (confetti, glowing effect, or a color burst). Only one animation runs at a time.

## Run locally

Option 1 (quickest): open `index.html` in your browser.

Option 2 (recommended): start a tiny local server from this folder.

If you have Python:
- `python -m http.server 5500`
- Open `http://localhost:5500`
